# 六爻卜卦

依《黃金策》15大類別進行占問分析的六爻占卜工具。

## 功能
- 起卦輸入（陽爻／陰爻／老陽／老陰）
- 自動計算農曆日干支、月干支
- 空亡、月破自動標示
- 伏神自動推算
- 15大占問類別：
  - 天時、年時、婚姻、求財、功名仕途、求職謀事
  - 行人音訊、出行遠行、疾病、訴訟官非、買賣交易
  - 求子嗣、宅舍居住、六畜牲口、尋人走失
- 每類別給出《黃金策》占斷結論

## 使用
直接開啟 `index.html`，或部署至 GitHub Pages。

## GitHub Pages 部署
1. 上傳 `index.html` 與 `README.md`
2. Settings → Pages → Branch: main → Save
3. 訪問 `https://<帳號>.github.io/<repo名>/`
